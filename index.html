About

The MYP ICT Knowledge Base is a simple, offline-friendly resource for students studying ICT in the Middle Years Programme.

How content is added

Teachers open app.js and add entries to the UNITS and TOPICS arrays — no coding knowledge needed beyond copy-paste. The admin panel also allows in-browser editing that saves to local storage.

Features

  • Browse lessons by curriculum unit
  • Full-text search across all content
  • Track which lessons you have visited (✓)
  • Dark mode
  • Print-friendly lesson pages
  • No internet, no install, no server required

Three files

  • index.html — page structure
  • style.css — visual design
  • app.js — all logic and content data
